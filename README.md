## Installation

1. Clone the repository

2. Install [asdf](https://github.com/asdf-vm/asdf) and plugins

``` bash
asdf plugin add nodejs
```

3. Run `asdf` install

```bash
asdf install
```

4. Install dependencies and start the development server

``` bash
npm ci
npm run dev
```
